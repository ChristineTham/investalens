# R1-P2: MVP Validation (Codespaces)

## Objective

Validate the complete R1 MVP in GitHub Codespaces: install dependencies, run database migrations, seed test data, run all tests, and verify the application works end-to-end.

## Environment

- GitHub Codespaces (Ubuntu)
- Node.js 20+, Python 3.11+
- PostgreSQL via Neon (provisioned in .env)

---

## Step 1: Environment Setup

```bash
# Install Node dependencies
npm install

# Verify TypeScript compilation
npx tsc --noEmit

# Run ESLint
npm run lint

# Run Prettier check
npx prettier --check .
```

---

## Step 2: Database Setup

```bash
# Create .env.local with DATABASE_URL from Neon dashboard
cp .env.example .env.local
# Edit .env.local with actual values

# Generate Prisma client
npx prisma generate

# Run migrations (creates all tables)
npx prisma migrate dev --name init

# Verify schema with Prisma Studio
npx prisma studio
```

---

## Step 3: Seed Test Data

**File to create if not exists: `prisma/seed.ts`**

Seed script should create:
- 1 test user (email: test@investalens.dev)
- 2 portfolios (Individual, SMSF)
- 10 ASX instruments (CBA, BHP, CSL, WES, TLS, FMG, ANZ, NAB, WBC, MQG)
- 30+ transactions across 6 months (BUY, SELL, DIVIDEND, SPLIT)
- 1 bond holding (FIIG bond)
- Cash account with matching settlements
- Custom group ("Sectors" with categories)
- Labels ("Growth", "Income", "Speculative")
- 1 API token for testing

```bash
npx prisma db seed
```
