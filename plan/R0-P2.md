## Troubleshooting

| Issue | Fix |
|-------|-----|
| Type errors in `db.ts` | Ensure `npx prisma generate` ran after schema expansion (R0-P1 Task 14) |
| Build fails on missing module | Check `pnpm ls [module]` — if missing, re-run R0-P0 Task 2 |
| CSS shows wrong colours | Confirm `globals.css` was replaced (not appended to) — should start with `@import "tailwindcss"` |
| `next/font` errors | Check layout.tsx imports `Noto_Sans`, `Noto_Serif`, `Noto_Sans_Mono` from `next/font/google` |
| Prisma validate fails | Check schema syntax — ensure no trailing commas or missing `@@id` |
| Dev server port conflict | Kill existing process: `lsof -ti:3000 | xargs kill` |
