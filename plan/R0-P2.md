# R0-P2: Project Setup (Validation Phase)

## Objective

Validate that the project scaffolding from R0-P1 is correct, all dependencies install, TypeScript compiles, and the dev server starts.

## Environment

- GitHub Codespaces (Ubuntu, Node.js 22, Python 3.12)
- All commands run in terminal

## Tasks

### 1. Install Dependencies

```bash
npm install
```

Verify: No errors. All packages resolve.

### 2. Generate Prisma Client

```bash
npx prisma generate
```

Verify: Prisma client generated without errors.

### 3. TypeScript Check

```bash
npx tsc --noEmit
```

Verify: No type errors. Fix any import issues.

### 4. ESLint Check

```bash
npm run lint
```

Verify: No errors (warnings acceptable). Fix any lint errors.

### 5. Build

```bash
npm run build
```

Verify: Next.js builds successfully. Fix any build errors.

### 6. Start Dev Server

```bash
npm run dev
```

Verify: Server starts on port 3000. Landing page renders "InvestaLens" heading.

### 7. Verify Python Functions

```bash
cd api && pip install -r requirements.txt && cd ..
```

Verify: numpy, scipy, pandas install without errors.

### 8. Verify DevContainer

Check that `.devcontainer/devcontainer.json` is valid JSON:

```bash
python -c "import json; json.load(open('.devcontainer/devcontainer.json'))"
```

### 9. Commit

```bash
git add -A
git commit -m "feat: R0 project scaffolding — Next.js, Prisma, Tailwind, shadcn/ui, Python functions"
```

## Success Criteria

- `npm install` completes without errors
- `npm run build` produces a successful build
- Dev server starts and serves the landing page
- No TypeScript compilation errors
- Python dependencies install correctly
- All configuration files are valid

## Troubleshooting

- If Prisma fails: check `DATABASE_URL` in `.env` — for local dev without a database, `prisma generate` still works (it only needs the schema file)
- If build fails on missing shadcn components: these are added in R1, the R0 build should only reference the landing page
- If Python deps fail: ensure Python 3.12+ is available; use `python3 -m pip install`
