# R3-P2: Multi-Market Validation (Codespaces)

## Objective

Validate multi-market features: international instruments, FX rates, multi-currency reports, international broker imports, and foreign tax calculations.

## Environment

- GitHub Codespaces (Ubuntu)
- Node.js 20+
- Open Exchange Rates API key configured in .env

---

## Step 1: FX Rate Setup

```bash
# Verify Open Exchange Rates API key
curl "https://openexchangerates.org/api/latest.json?app_id=$OPEN_EXCHANGE_RATES_APP_ID"

# Run FX rate fetch
curl http://localhost:3000/api/cron/fx-rates

# Verify rates stored in DB
npx prisma studio
# Check ExchangeRate table has entries
```

---

## Step 2: Seed International Data

Add to seed script:
- 5 US instruments (AAPL, MSFT, GOOGL, AMZN, TSLA)
- 3 UK instruments (SHEL.L, AZN.L, HSBA.L)
- 2 international ETFs held on ASX (VGS, IVV)
- Transactions in USD, GBP currencies
- FX rates for AUD/USD, AUD/GBP for transaction dates

```bash
npx prisma db seed
```

## Step 3: Functional Testing

### Multi-Market Instruments
- [ ] Search finds US stocks (no suffix needed in search)
- [ ] Search finds UK stocks (returns .L suffix)
- [ ] Price fetch works for US (AAPL → AAPL)
- [ ] Price fetch works for UK (SHEL → SHEL.L)
- [ ] Price fetch works for HK (0005 → 0005.HK)
- [ ] Exchange registry returns correct metadata

### FX Rates
- [ ] Latest rate fetch returns valid AUD/USD rate
- [ ] Historical rate fetch for specific date works
- [ ] Cross-rate calculation correct (e.g., GBP/EUR via USD)
- [ ] Cached rates used when available
- [ ] Rate staleness detection (>24h old)

### Multi-Currency Valuation
- [ ] Portfolio value sums correctly across currencies
- [ ] Each holding shows both local and AUD value
- [ ] Currency gain/loss calculated separately
- [ ] Performance report splits local vs currency return
- [ ] Total return = local return × currency return

### International Import
- [ ] Interactive Brokers CSV imports correctly
- [ ] Schwab CSV handles date format (mm/dd/yyyy)
- [ ] Degiro CSV handles comma decimal separator
- [ ] Trading212 CSV applies exchange rate from file
- [ ] Market auto-detection from ticker symbol
- [ ] Currency auto-assignment from exchange

### Foreign Tax
- [ ] US dividend withholding = 15% (DTA rate)
- [ ] UK dividend withholding = 15%
- [ ] HK dividend withholding = 0%
- [ ] Foreign income report groups by country
- [ ] Foreign tax credits calculated correctly
- [ ] International CGT uses FX at buy date for cost base
- [ ] Forex gain separated from capital gain

### International Corporate Actions
- [ ] Spin-off creates new holding with cost base allocation
- [ ] ADR conversion adjusts quantity and code
- [ ] Stock dividend adds shares at FMV

### ETF X-ray (International)
- [ ] VGS shows US mega-cap holdings
- [ ] IVV shows S&P 500 top 10
- [ ] Currency exposure shows USD for VGS, AUD for VGAD
