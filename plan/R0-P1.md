- R0-P0 completed (shadcn CLI ran, all deps installed, Prisma client generated)
- Reference: `DESIGN.md` for Rosely design system
- Reference: `docs/ARCHITECTURE.md` for tech stack decisions

- **shadcn** — Theming, component composition, Base UI
- **next-best-practices** — File conventions, App Router, RSC boundaries
- **tailwind-design-system** — Design tokens, CSS variable setup
- **nextauth-authentication** — Auth.js v5 patterns
- **env-vars** — Environment variable conventions

```typescript
export function formatCurrency(
  amount: number,
  currency: string = "AUD",
  locale: string = "en-AU"
): string {
  return new Intl.NumberFormat(locale, {
    style: "currency",
    currency,
    minimumFractionDigits: 2,
    maximumFractionDigits: 2,
  }).format(amount);
}

export function formatPercent(value: number, decimals: number = 2): string {
  return `${value >= 0 ? "+" : ""}${value.toFixed(decimals)}%`;
}

export function formatDate(date: Date | string, format: string = "short"): string {
  const d = typeof date === "string" ? new Date(date) : date;
  if (format === "iso") return d.toISOString().split("T")[0];
  return d.toLocaleDateString("en-AU", {
    year: "numeric",
    month: format === "long" ? "long" : "short",
    day: "numeric",
  });
}
```

### 6. Create `src/lib/db.ts`

Uses the Prisma driver adapter pattern with `@prisma/adapter-pg` and `pg`:

```typescript
import { PrismaClient } from "../generated/prisma/client";
import { PrismaPg } from "@prisma/adapter-pg";
import pg from "pg";

const globalForPrisma = globalThis as unknown as {
  prisma: PrismaClient | undefined;
};

function createPrismaClient() {
  const pool = new pg.Pool({ connectionString: process.env.DATABASE_URL });
  const adapter = new PrismaPg(pool);
  return new PrismaClient({
    adapter,
    log: process.env.NODE_ENV === "development" ? ["query"] : [],
  });
}

export const db = globalForPrisma.prisma ?? createPrismaClient();

if (process.env.NODE_ENV !== "production") globalForPrisma.prisma = db;
```

> **Note:** With the new `prisma-client` generator, the import path is relative to the `output` directory specified in `schema.prisma`. The driver adapter (`PrismaPg`) is required at runtime.

### 8. Create `vercel.json`

Source: https://vercel.com/docs/functions/runtimes/python

> Note: Python is a built-in Vercel runtime — no `runtime` field needed. Vercel auto-detects Python functions from `/api/*.py` files that export a `handler` class or ASGI/WSGI `app`. Dependencies are read from `pyproject.toml` or `requirements.txt`.

```json
{
  "$schema": "https://openapi.vercel.sh/vercel.json",
  "functions": {
    "api/**/*.py": {
      "maxDuration": 60,
      "excludeFiles": "{tests/**,__tests__/**,**/*.test.py,**/test_*.py}"
    }
  }
}
```

- Use `@base-ui-components/react` (Base UI) — no Radix packages

- All imports use the `@/` path alias
