# R4-P2: Platform Maturity Validation (Codespaces)

## Objective

Validate all R4 features: net worth, webhooks, PDF export, Sharesight integration, notifications, performance, audit log, dark mode, and mobile responsiveness.

## Environment

- GitHub Codespaces (Ubuntu)
- Node.js 22+, Python 3.12
- Resend API key configured, Sharesight partner credentials (if available)

## Recommended Skills

Invoke these skills for best-practice guidance during this phase:

- **prisma-cli** — Migration for net worth, webhook, audit log tables
- **verification** — End-to-end user journey testing
- **vercel-optimize** — Performance benchmarking, bundle analysis
- **next-browser** — Dark mode rendering verification, component inspection
- **web-design-guidelines** — Mobile responsiveness audit, WCAG AA check
- **deploy-to-vercel** — Final production deployment
