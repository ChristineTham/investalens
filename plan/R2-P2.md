# R2-P2: Advanced Analytics Validation (Codespaces)

## Objective

Validate all R2 analytics features in GitHub Codespaces: Python functions, optimisation, Monte Carlo, backtesting, AI features, and tools.

## Environment

- GitHub Codespaces (Ubuntu)
- Node.js 22+, Python 3.12
- All R1 + R2 code deployed

## Recommended Skills

Invoke these skills for best-practice guidance during this phase:
- **verification** — End-to-end flow testing (TS → Python → response)
- **prisma-cli** — Database operations for test data
- **next-browser** — Debug chart rendering, component tree inspection

## Step 3: Functional Testing

### Backtesting
- [ ] Equal weight backtest produces equity curve
- [ ] Mean-variance backtest shows different weights per period
- [ ] Walk-forward analysis shows IS vs OOS performance
- [ ] Strategy comparison ranks models correctly
- [ ] Backtest results are cached

### Optimisation
- [ ] Max Sharpe returns valid weights summing to 1
- [ ] Min Variance weights favour low-volatility assets
- [ ] HRP produces stable weights without return estimation
- [ ] Risk Parity distributes risk equally
- [ ] Constraints (min/max weight) are respected
- [ ] Efficient frontier generates smooth curve
- [ ] Black-Litterman shifts weights toward views

### Monte Carlo
- [ ] Bootstrap MC produces fan chart data
- [ ] Parametric MC uses multivariate normal
- [ ] Copula MC captures tail dependence
- [ ] Statistics (VaR, CVaR, prob of loss) are correct
- [ ] Histogram data has 50 bins
- [ ] 10,000 simulations complete within timeout

### Estimation
- [ ] All 5 return estimators produce results
- [ ] All 11 covariance estimators produce positive-definite matrix
- [ ] Distribution fitting returns AIC/BIC rankings
- [ ] Copula fitting returns tail dependence

### Factor Analysis
- [ ] Fama-French regression produces alpha + 3 betas
- [ ] PCA extracts components with scree plot data
- [ ] Factor model explains >70% of variance

### Tactical Allocation
- [ ] Momentum strategy ranks assets correctly
- [ ] Mean reversion overweights recent losers
- [ ] Vol targeting inversely weights volatility

### Stress Testing
- [ ] Historical scenarios return portfolio impact
- [ ] Custom shocks propagate correctly
- [ ] Assets with high weights contribute most to loss

### Cross-Validation
- [ ] Walk-forward CV produces valid OOS metrics
- [ ] Combinatorial Purged CV shows no leakage
- [ ] Model ranking is consistent

### AI Features
- [ ] AI Import parses a sample broker PDF
- [ ] Confidence scores are reasonable (>0.7)
