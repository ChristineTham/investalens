# R2-P0: Codespaces Environment Setup

## Objective

Prepare the GitHub Codespaces environment for R2 implementation: install Python analytics packages, verify the Python function runtime, seed benchmark data, and confirm the TypeScript ↔ Python bridge works.

## Prerequisites

- R1 complete (working MVP with holdings, prices, reports)
- Codespace running (Ubuntu, Node.js 22, Python 3.12)

---

## Step 1: Python Dependencies

```bash
# Ensure pyproject.toml has all required packages (add if missing)
uv add skfolio scipy numpy pandas statsmodels

# Sync the environment
uv sync

# Verify imports
uv run python -c "
import skfolio; print(f'skfolio {skfolio.__version__}')
import scipy; print(f'scipy {scipy.__version__}')
import numpy; print(f'numpy {numpy.__version__}')
import pandas; print(f'pandas {pandas.__version__}')
import statsmodels; print(f'statsmodels {statsmodels.__version__}')
print('All analytics packages OK')
"
```

## Step 5: Benchmark Price Data Seed

Seed benchmark instrument records so the analytics data service can fetch benchmark prices alongside portfolio data:

```bash
# Run a seed script that creates benchmark instruments
pnpm exec tsx scripts/seed-benchmarks.ts
```

**File: `scripts/seed-benchmarks.ts`**

Creates `Instrument` records for benchmark indices:

| Code | Name | Market | Type |
|------|------|--------|------|
| ^AXJO | S&P/ASX 200 | ASX | INDEX |
| ^AXJOA | S&P/ASX 200 Accumulation | ASX | INDEX |
| ^GSPC | S&P 500 | NYSE | INDEX |
| ^MSCI | MSCI World | GLOBAL | INDEX |
| STW.AX | SPDR S&P/ASX 200 Fund | ASX | ETF |
| SPY | SPDR S&P 500 ETF | NYSE | ETF |

These serve as benchmark selectors for risk metrics, backtesting, and comparison charts.
